\# QUIZ CHEAT



\## Features

\- OCR-based question extraction

\- AI-powered answer selection

\- Two-stage reasoning system



\## Setup

1\. Clone repo

2\. Install dependencies:

&#x20;  pip install -r requirements.txt



3\. Create .env:

&#x20;  GEMINI\_API\_KEY=your\_key



4\. Run:

&#x20;  python main.py



\## Controls

F6: Toggle question box  

F7: Toggle options box  

F8: Solve question
